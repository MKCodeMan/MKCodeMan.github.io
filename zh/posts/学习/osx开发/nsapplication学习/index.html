NSApplication学习

四月 15, 2024 One-minute read
Mac开发 • 学习

NSApplication是Cocoa框架中的一个类，用于创建和管理Mac应用的主事件循环和事件处理，是macOS应用的基础。

App无storyboard启动 Link to heading

1.首先删除main.storyboard文件，然后再Build Setting中AppKit Main Storyboard File Base Name清空。 2.创建main.swift文件 3.移除AppDelegate中*@main* 4.在mian中添加代码

let app = NSApplication.shared

let delegate = AppDelegate()
app.delegate = delegate

app.run()

5.在AppDelegate中配置应用的菜单和窗口等
