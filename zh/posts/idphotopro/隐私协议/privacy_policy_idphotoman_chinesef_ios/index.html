隐私协议

四月 1, 2024 One-minute read
IdPhotoMan

感謝您使用我們的應用程式。 在我們的應用程式中，我們尊重和保護用戶的隱私。 本隱私權政策解釋了我們如何收集、使用和揭露使用者的個人資訊。 請您注意，本協議僅適用於軟體“證件照專家”，不適用於其他第三方提供的服務。 請您在使用人工智慧證件照產品及服務前，請務必閱讀並充分理解本協議，特別是粗體/底線文字標註的條款。 在開始使用之前，請仔細閱讀並確保您完全理解並同意條款和條件。 您的使用行為即構成您對本協議的充分理解和同意。 如果您對本協議主題有任何疑問、意見或建議，請透過APP提供的各種聯絡方式與我們聯絡。

定義 Link to heading

  • 證件照專家：證件照專家行動應用客戶端。
  • 個人資訊：是指以電子或其他方式記錄的能夠單獨或與其他資訊結合識別特定自然人身分或反映特定自然人活動情況的任何類型的資訊。
  • 個人敏感資訊：指個人訊息，包括身分證號碼、個人生物辨識資訊、銀行帳號、財產資訊、位置資訊、交易資訊、兒童（含18歲以下）資訊等。
  • 個人資訊刪除：是指在與實現產品服務功能相關的系統中刪除個人訊息，並將其維持在無法檢索或存取的狀態。
  • 匿名化處理：是指對個人資訊進行技術處理，使個人資訊主體無法被辨識、處理後的資訊無法恢復的過程。
  • SDK：指軟體工具開發包。

我們擁有的權限以及您個人資訊的收集和使用 Link to heading

我們僅在法律規定合法且必要的情況下以及在您使用我們的產品和/或服務時收集和使用您的個人資訊。 使用它是為了：提供我們產品和/或服務的基本功能，收集和使用必要的資訊；您必須允許我們這樣做。 如果您拒絕提供相應訊息，您將無法成功使用我們的產品和/或服務。

  • 允許應用程式讀取手機硬碟的內容，該權限用於拍照和製作證件照。
  • 允許應用程式修改或刪除手機硬碟中的內容，該權限用於儲存證件照功能。
  • 取得相簿的讀寫權限，此權限用於上傳照片並建立證件照。
  • 取得臉部數據，該數據將僅被演算法用於建立證件照，不會與第三方共享該數據，並且不會被儲存；
  • 取得有關設備的信息，例如設備型號和作業系統版本。
  • 取得UUID。 此權限可用於提供個人化的服務。

徵得授權同意的例外 Link to heading

根據相關法律法規規定，以下情形中收集您的個人資訊無需徵得您的授權同意：

  • 與國家安全、國防安全有關的；
  • 與公共安全、公共衛生、重大公共利益有關的；
  • 與犯罪偵查、起訴、審判和判決執行等有關的；
  • 出於維護個人資訊主體或其他個人的生命、財產等重大合法權益但又很難得到您本人同意的；
  • 所收集的個人資訊是您自行向社會大眾公開的；
  • 從合法公開揭露的資訊中收集個人資訊的，如合法的新聞報告、政府資訊公開等管道；
  • 根據您的要求簽訂合約所必需的；
  • 用於維護所提供的產品或服務的安全穩定運作所必需的，例如發現、處置產品或服務的故障；
  • 為合法的新聞報道所必需的；
  • 學術研究機構基於公共利益開展統計或學術研究所必要，且對外提供學術研究或描述的結果時，對結果中所包含的個人資訊進行去識別化處理的；
  • 法令規定的其他情形。

個人資訊的存儲 Link to heading

本App僅在為實現目的所必需的時間及法律行政法規所規定得保存期限內保留您的個人信息，您上傳的照片僅在服務器存儲1天，照片的臉部信息僅用於摳圖，不 會進行存儲，您上傳的照片一天后會自動刪除，製作好的證件照僅會存儲在本地。

我們如何分享、轉讓、公開揭露您的個人資訊 Link to heading

請您知悉，我們僅會出於合法、正當、必要、特定、明確的目的共享您的個人信息，對我們僅為實現本政策中聲明的目的與之共享個人信息的公司和組織，我們會要求 他們嚴格遵守我們協議並採取相關的安全措施來保護您的個人資訊。 當我們發生被收購、與第三方合併等事由時，如涉及個人資訊的轉讓，我們會要求受讓您個人資訊的公司、組織繼續接受本協議的約束，否則，我們將要求該公司、組織重新 取得您的授權。

資訊的收集與使用 Link to heading

我們的應用程式不會收集、儲存或分享用戶的個人信息，除非用戶自願提供。 當使用者使用我們的應用程式時，我們可能會要求某些個人信息，例如存取照片庫或相機權限，但此資訊僅用於創建身份照片，不會由我們收集或儲存。

安全 Link to heading

我們已經實施了適當的安全措施，保護使用者提供的個人資訊免受未經授權的存取、揭露、變更或銷毀。

兒童隱私 Link to heading

我們不會有意從18歲以下的兒童收集個人資訊。 如果您是18歲以下兒童的父母或監護人，並且認為您的孩子可能向我們提供了個人訊息，請與我們聯繫，我們將採取適當措施刪除此類資訊。

更新 Link to heading
