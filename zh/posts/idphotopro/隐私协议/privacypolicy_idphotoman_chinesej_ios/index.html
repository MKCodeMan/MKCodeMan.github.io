隐私协议

四月 1, 2024 One-minute read
IdPhotoMan

隐私政策 Link to heading

感谢您使用我们的应用程序。在我们的应用程序中，我们尊重和保护用户的隐私。本隐私政策解释了我们如何收集、使用和披露用户的个人信息。

信息的收集和使用 Link to heading

我们的应用程序不会收集、存储或分享用户的个人信息，除非用户自愿提供。当用户使用我们的应用程序时，我们可能会请求某些个人信息，例如访问照片库或相机权限，但此信息仅用于创建身份照片，不会由我们收集或存储。

第三方服务 Link to heading

我们的应用程序使用谷歌的 Firebase 服务提供某些功能，并使用 AdMob 服务显示广告。这些服务可能会收集关于您对应用程序的使用情况，如使用数据、设备信息和位置信息。请参阅谷歌的隐私政策获取更多信息：

Firebase 隐私政策

AdMob 隐私政策

安全 Link to heading

我们已经实施了适当的安全措施，保护用户提供的个人信息免受未经授权的访问、披露、更改或销毁。

儿童隐私 Link to heading

我们不会有意从18岁以下的儿童那里收集个人信息。如果您是18岁以下儿童的父母或监护人，并且认为您的孩子可能向我们提供了个人信息，请与我们联系，我们将采取适当措施删除此类信息。

更新 Link to heading

我们可能会不时更新本隐私政策。我们建议您定期查看此页面，了解任何变更。对于重大变更，我们还可能向您提供通知（例如，通过电子邮件或应用内通知）。
